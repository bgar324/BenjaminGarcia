# OG Image Guide

## Current Status
A basic OG image placeholder has been created at `/public/static/og-image.png` (1200x630px) using your profile photo.

## Recommended Improvements
For a more professional OG image, consider creating a custom design with:

### Option 1: Figma/Canva Template
- Dimensions: 1200x630px
- Include: Your name, title, and optionally your profile photo
- Brand colors: Use colors from your portfolio (grays, blacks, accent colors)
- Typography: Cabinet Grotesk font if possible
- Export as PNG at 1200x630px

### Option 2: Use a Service
- [OG Image Generator](https://og-playground.vercel.app/)
- [Social Image](https://www.socialimage.ai/)
- [Bannerbear](https://www.bannerbear.com/)

### Design Tips
- Keep text large and readable (minimum 48px font size)
- Use high contrast
- Avoid small details (they get lost when scaled down)
- Test how it looks on Twitter, LinkedIn, and Slack

## Replacing the Placeholder
Simply replace `/public/static/og-image.png` with your custom design (keep the filename and dimensions).
